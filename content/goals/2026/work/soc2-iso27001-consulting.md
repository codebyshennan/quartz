## Goal
Build a viable freelance service line under Fracxional for SOC 2 readiness and ISO 27001 implementation consulting — targeting SEA startups doing enterprise sales unlocks.

**Positioning:** Ex-Partior DevSecOps + Iterative VC network → credible, technical, not just policy-writers.

**Revenue target:** First paid engagement by Month 3 ($5–10K SGD). Scale or reposition decision by Month 6.

## Key results

### Month 1 — Foundation
- [ ] Enroll in ISO 27001 Lead Implementer course — **PECB eLearning recommended ($489 USD via Smatica), BSI SG classroom alt ($2,800–3,500 SGD, SkillsFuture-eligible)**
- [ ] Audit own knowledge gaps: SOC 2 Trust Service Criteria, ISO Annex A controls
- [ ] Read 2–3 real SOC 2 Type I reports (get from network or public sources)
- [ ] Define service packages: "SOC 2 Readiness in 90 days" + "ISO 27001 ISMS Setup"
- [ ] Draft a one-pager for Fracxional positioning

### Month 2 — Credibility Building
- [ ] Complete ISO 27001 Lead Implementer exam
- [ ] Identify 2–3 Iterative portfolio companies that need SOC 2
- [ ] Offer first engagement at reduced rate in exchange for case study rights
- [ ] Build reusable assets: gap assessment template, policy library, evidence tracker

### Month 3 — First Paid Engagement
- [ ] Land first paid client (target: $5–10K SGD for a Type I readiness project)
- [ ] Deliver gap assessment + control design + policy documentation
- [ ] Document the repeatable playbook as you go
- [ ] Update Fracxional website/LinkedIn to reflect the service line

### Month 4 — Productize
- [ ] Package playbook into fixed-scope offerings with clear deliverables + timelines
- [ ] Build templated artifacts: risk register, SoA, vendor questionnaire, evidence folder
- [ ] Explore partnerships with CPA firms (SOC 2) and CBs (ISO cert)
- [ ] Target MAS/BSP-regulated fintechs in SEA as secondary pipeline

### Month 5 — Pipeline & Referrals
- [ ] Warm intros via Iterative portfolio
- [ ] Publish 1–2 LinkedIn posts positioning as SEA security consultant
- [ ] Aim for 2nd paid engagement — ideally ISO 27001 to diversify
- [ ] Explore subcontracting to larger consulting firms as associate

### Month 6 — Review & Scale Decision
- [ ] Assess revenue vs time invested
- [ ] Decide: scale as standalone service line vs fold into Fracxional
- [ ] If viable: set next 6-month revenue target, hire first subcontractor
- [ ] If not: pivot positioning or deprioritize

## Target client profile
- SEA startup, Series A–B, doing enterprise sales into corporates/gov
- Needs SOC 2 Type I for deal unlock or ISO 27001 for MAS/enterprise vendor approval
- 20–100 person company, no internal security function
- Budget: $10–30K SGD per engagement

## Resources
- PECB ISO 27001 Lead Implementer: https://pecb.com
- AICPA SOC 2 Trust Service Criteria
- Drata / Vanta playbooks (study their self-serve tooling to understand market)
